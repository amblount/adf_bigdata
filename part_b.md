### Relational Databases
This lesson introduced relational databases. If you are completely new to databases, you may find it useful to review the content in the Database Fundamentals course on Microsoft Virtual Academy at https://mva.microsoft.com/en-US/training-courses/sql-database-fundamentals-16944.

### Data Warehousing
Data warehousing is a specific database workload for large volumes of historical data that can be used for reporting and analysis. 
If you've never worked with a star-schema data warehouse, you may find it useful to audit course 
- DAT216x: Delivering a Relational Data Warehouse on edx at https://aka.ms/edx-dat216x-about  (note that this course is not a requirement for the Microsoft Professional Program in Big data, but it is recommended as a pre-requisite for the course DAT220x: Delivering a Data Warehouse in the Cloud).
